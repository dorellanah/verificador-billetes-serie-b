# Verificador Serie B - Billetes BCB 🇧🇴

Aplicación web estática para verificar si un número de serie de la **Serie B** se encuentra dentro de los rangos publicados oficialmente.

---

## 🌐 Demo

Disponible en GitHub Pages:

https://TU-USUARIO.github.io/TU-REPO/

---

## ⚙️ ¿Qué hace esta aplicación?

- Permite ingresar un número de serie manualmente o capturarlo desde la cámara.
- Utiliza reconocimiento óptico de caracteres (OCR) para leer números de serie desde imágenes.
- Verifica si el número pertenece a los rangos oficiales publicados para la **Serie B**.
- Funciona completamente en el navegador.
- No envía datos a ningún servidor.
- No almacena información.

---

## ⚠️ Descargo de Responsabilidad

Esta aplicación utiliza reconocimiento óptico de caracteres (OCR) para leer números de serie desde imágenes capturadas por la cámara del dispositivo.

La verificación se limita exclusivamente al número de serie correspondiente a la **Serie B**.  
No se analizan elementos de seguridad físicos ni características oficiales del billete.

La aplicación **no garantiza la autenticidad del billete** ni determina si el mismo es falso.

La precisión del reconocimiento **no está garantizada**. Los resultados pueden contener errores debido a factores como iluminación, calidad de imagen, enfoque, estado del billete o limitaciones propias del OCR.

La información generada debe considerarse únicamente como **referencia orientativa**.  
Ante cualquier duda, el usuario debe verificar manualmente el número en el sitio oficial del Banco Central de Bolivia:

https://www.bcb.gob.bo/?q=content/verificador-de-n%C3%BAmero-de-serie
